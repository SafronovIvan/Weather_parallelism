Weather Analyzer

Программа Weather Analyzer асинхронно собирает данные о погоде для списка городов через бесплатное API Open-Meteo. Она автоматически определяет координаты по названию города и загружает текущую погоду, ветер и прогноз.

Установка и запуск
Для работы требуется установить библиотеку aiohttp: pip install aiohttp. Запуск: python request_city_info.py. Файлы конфигурации не нужны.

Использование
Измените 77 строку, добавив больше городов:
save_to_json(asyncio.run(get_weather_by_cities(["Saint Petersburg", "London"])))

Что содержится в данных
Для каждого города возвращаются: название, координаты, текущая температура, скорость и направление ветра, код погоды с переводом, а также почасовой и дневной прогноз.

Сохранение результатов
Сохранить данные в JSON можно функцией save_weather_to_json(weather_data, "weather.json").

Используемые API
Геокодирование: https://geocoding-api.open-meteo.com/v1/search
Погода: https://api.open-meteo.com/v1/forecast
